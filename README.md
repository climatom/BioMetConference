# International Congress of Biometeorology Workshop

## Preparing for the Climate Data Analysis Workshop

Thank you for registering for the workshop. To ensure we can spend as much time as possible on the science during the session, please complete the following short setup before attending.

The setup should take approximately **10 minutes**.

### 1. Create a Google account (if required)

Google Colab requires a Google account.

If you already use Gmail, Google Drive, or any other Google service, you already have one and can skip this step.

Otherwise, create a free account here:

https://accounts.google.com/signup

### 2. Open the workshop notebook

Open the following link in your web browser:

**[[Workshop Colab notebook link]](https://colab.research.google.com/github/climatom/BioMetConference/blob/main/notebooks/00_test_environment.ipynb)**

This will launch the notebook in Google Colab.

### 3. Sign in

If prompted, sign in using your Google account.

### 4. Save your own copy

Once the notebook has opened, select:

**File → Save a copy in Drive**

This creates your own editable copy in your Google Drive.

Please work from **your own copy**, not the original shared notebook.

## 5. Run the notebook

Run each code cell from top to bottom.

You can either:

* click the ▶ button next to each cell, or
* select **Runtime → Run all**.

The first cell installs the required Python packages and may take a minute or two to complete.

### 6. Check that everything works

The notebook will:

* install the required software,
* download a small weather forecast,
* load the data,
* produce a simple figure.

If you reach the final cell and see the success message without any errors, your environment is ready for the workshop.

### 7. If you encounter a problem

If the notebook stops with an error that you cannot resolve, please email me **before the workshop** with:

* a screenshot of the error message,
* the name of your web browser,
* and (if possible) the notebook link.

This will allow us to resolve any issues in advance.

---

## On the day of the workshop

Please bring:

* a laptop (Windows, macOS or Linux),
* a modern web browser (Chrome, Edge, Firefox or Safari),
* your Google account login details,
* a reliable internet connection,
* and (optionally) a charger if attending in person.

No software needs to be installed on your computer.
